Node.js & npm installation was required.

Steps I followed:

# Step 2: Navigate to the project folder
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Start the dev server with hot reload and preview
npm run dev
